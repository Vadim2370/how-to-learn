## О проекте
Одностраничный сайт c информацией о методах обучения и ссылками на ресурсы об обучении.

## Технологии

* Семантические теги
### СSS

* Псевдоклассы CSS
* Позиционирование блоков и элементов
* Анимация и трансформация элементов страницы посредством CSS
* Iframe
* API Youtube
